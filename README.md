# Full-Stack Application Development Course (Web & Mobile) — Beginner Level

## About the course

This course introduces how modern applications are built and how backend, web, and mobile parts work together. Over **10 weeks**, you learn fundamentals of backend development, web frontend development, and mobile development, including how those layers communicate.

The focus is a **basic but complete application**, built step by step: client–server communication, APIs, databases, and application structure. Instruction is **project-based**—one main application grows as new topics are introduced.

## Who this course is for

- Beginners who want an introduction to software development  
- University students or graduates from non-technical or semi-technical backgrounds  
- Professionals who want practical insight into how applications are built  
- Entrepreneurs or product owners who want to understand development workflows  
- Learners with basic English who can follow explanations and read simple technical text  

**No prior programming experience is required.** You should be comfortable using a computer and practicing regularly. Curiosity, patience, and consistency matter more than a technical background.

## What you will gain (learning objectives)

After successfully completing the course, you should be able to:

- Understand the basic structure of a modern app (backend, frontend, mobile)  
- Create a simple backend with **JavaScript (Node.js)** and basic **REST** APIs  
- Work with a **relational database** at a basic level and see how data is stored  
- Build simple web UIs with **React** and connect them to a backend API  
- Build a basic **React Native** mobile app and connect it to the same backend  
- Send and receive data across backend, web, and mobile  
- Understand common errors and use tools to investigate and fix basic issues  
- Use **Git** at a basic level to track changes  
- Use **AI tools** to support understanding and debugging while staying in control of logic  
- Write clearer, more organized code for small applications  
- Explain at a basic level how your app works and how the pieces connect  
- Know how to keep learning after the course ends  

By the end, you will have practical exposure to full-stack development and a clearer path for what to study next. The course targets **confident beginners with a solid foundation**, not advanced specialists.

## Programme content

The course blends guided instruction, hands-on practice, and project work: example code, supervised exercises, and a shared application built gradually. Emphasis is on practical understanding, tools, and steady skill growth.

## Course schedule

### Week 1 — Basics of software development and web technologies

**Topics:** What software is; how web apps work; HTML, CSS, and simple JavaScript; setting up **Cursor** and related tools.

**Activities:** Simple HTML page; basic CSS; short JavaScript examples; IDE setup.

### Week 2 — Programming fundamentals and version control basics

**Topics:** Variables, conditions, loops; JavaScript practice; common errors and debugging; **Git** concepts.

**Activities:** Guided coding exercises; Git repository; commits and tracking changes.

### Week 3 — Databases and basic data operations

**Topics:** Why apps use databases; tables and records; basic **SQL**; connecting a backend to a database; simple create/read operations.

**Activities:** SQL practice; connect backend to DB; **Project Phase 1 (Database)**.

### Week 4 — Backend fundamentals and API concepts

**Topics:** Role of a backend server; simple server; backend–database connection; API routes; testing with **Postman**; requests and responses.

**Activities:** Build and connect a simple server; define API routes with guidance; test in Postman.

### Week 5 — Frontend development fundamentals

**Topics:** **React** concepts; components; props and state; simple UI; fetching from the backend API.

**Activities:** Component exercises; call backend from the UI; **Project Phase 2 (Backend)**.

### Week 6 — Web application structure and basic authentication

**Topics:** Multi-page structure; routing; login and basic auth flow; simple protected pages; UI improvements.

**Activities:** Routing practice; auth demonstration; extend UI with exercises.

### Week 7 — Mobile application development fundamentals

**Topics:** **React Native** components; mobile screen with guidance; connect mobile app to backend; web vs mobile.

**Activities:** Guided mobile screen; API usage from mobile; **Project Phase 3 (Frontend)**.

### Week 8 — Integration of backend, web, and mobile

**Topics:** One backend serving web and mobile; user input across clients; integration debugging; end-to-end flow.

**Activities:** Integrate clients with shared backend; **Project Phase 4 (Mobile app)**.

### Week 9 — Code organization and AI-assisted development

**Topics:** Code structure; readable vs unclear code; how AI tools support development; small refactors with practice.

**Activities:** Organization review; AI-assisted improvements on small sections.

### Week 10 — Guided full-stack project completion and review

**Topics:** Finish the guided full-stack project; review features; prepare a short explanation; final demonstration.

**Activities:** Complete project; present main features.
